nor-timezones
=============

Exports static array of timezones

Install
-------

`npm install nor-timezones`

Usage
-----

```javascript
var timezones = require('nor-timezones');
```

..and `timezones` will be an array of strings like:

```
[
  "Africa/Abidjan",
  "Africa/Accra",
  "Africa/Addis_Ababa",
....
  "US/Mountain",
  "US/Pacific",
  "US/Samoa",
  "WET"
]
```
